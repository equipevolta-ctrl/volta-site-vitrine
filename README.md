# 🚀 Volta Site Vitrine

Bienvenue sur le dépôt du site vitrine d'**Équipe Volta** !

## 📝 Description

Ce projet contient le site vitrine officiel d'Équipe Volta, présenté en HTML pur pour une performance optimale et une compatibilité maximale.

## 🎯 Objectifs

- Présenter l'équipe Volta et ses projets
- Fournir une vitrine professionnelle et moderne
- Démontrer notre expertise en développement
- Faciliter le contact et la collaboration

## 🛠️ Technologies

- **HTML5** : Structure sémantique et moderne
- **CSS3** : Styling responsive et élégant
- **JavaScript** : Interactivité et dynamisme
- **GitHub Pages** : Hébergement gratuit et automatisé

## 📦 Structure du projet

```
volta-site-vitrine/
├── index.html          # Page d'accueil principale
├── styles/             # Feuilles de style CSS
├── scripts/            # Fichiers JavaScript
├── images/             # Ressources graphiques
└── README.md          # Documentation du projet
```

## 🌐 Accès au site

Le site est automatiquement publié via **GitHub Pages** :
- **URL** : https://equipevolta-ctrl.github.io/volta-site-vitrine/

Les modifications apportées à la branche `main` sont déployées automatiquement.

## 🚀 Démarrage rapide

### Localement
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/equipevolta-ctrl/volta-site-vitrine.git
   ```
2. Ouvrez `index.html` dans votre navigateur

### Pour éditer
- Modifiez les fichiers HTML, CSS ou JavaScript
- Committez et poussez vos changements sur `main`
- Attendez quelques secondes pour que GitHub Pages redéploie le site
